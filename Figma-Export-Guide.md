# 时光印记 Figma 导入指南

## 推荐方式

使用 Figma 插件 `html.to.design` 导入本地页面。

本地服务地址：

```text
http://127.0.0.1:5173/
```

推荐画板尺寸：

```text
393 x 852
```

这是当前项目的 iPhone 展示容器尺寸，适合面试演示和作品集展示。

## 导入顺序

1. 开屏页

```text
http://127.0.0.1:5173/
```

用途：展示产品定位、核心路径和 Demo 入口。

2. 时光机首页

```text
http://127.0.0.1:5173/home
```

用途：展示案例池、本地/全部区域切换、发布入口。

3. 发现页

```text
http://127.0.0.1:5173/explore
```

用途：展示搜索、横向标签筛选、双列卡片流。

4. 发布流程

```text
http://127.0.0.1:5173/publish
```

用途：展示上传图片/文字描述、补充信息、分享卡、确认发布。

5. 详情页

```text
http://127.0.0.1:5173/missing/demo-classmate-li
```

用途：展示线索详情、结构化信息、私聊提供线索入口。

6. 消息页

```text
http://127.0.0.1:5173/messages
```

用途：展示用户收到线索反馈后的消息入口。

7. 私聊页

```text
http://127.0.0.1:5173/chat/demo-old-photo/demo-publisher
```

用途：展示线索沟通链路。

8. 我的页面

```text
http://127.0.0.1:5173/profile
```

用途：展示我的寻人、通知中心、已找到档案、会员中心等二级入口。

9. 我的寻人

```text
http://127.0.0.1:5173/my-missing
```

用途：展示发布者视角的寻人管理，匹配数仅在这里保留。

## Figma 操作步骤

1. 打开 Figma，新建文件。
2. 安装或打开 `html.to.design` 插件。
3. 选择 `Import web page by URL`。
4. 逐个粘贴上面的本地 URL。
5. 每个页面导入后命名为：

```text
01 开屏页
02 时光机首页
03 发现页
04 发布流程
05 线索详情
06 消息页
07 私聊页
08 我的页面
09 我的寻人
```

## 作品集摆放建议

建议在 Figma 中分三组摆放：

```text
产品入口：开屏页 / 时光机首页 / 发现页
核心链路：发布流程 / 线索详情 / 私聊页
管理闭环：消息页 / 我的页面 / 我的寻人
```

## 验收重点

- `+` 发布按钮只出现在时光机和发现两个一级栏目。
- 卡片第一信息是人名或宠物名，地点和时间次之。
- 公开卡片不展示浏览数和匹配数。
- 带图发布后，图片能在时光机/发现池中显示。
- 详情页能显示内置案例和用户发布记录。
- 二级/三级页面不出现发布悬浮按钮。
